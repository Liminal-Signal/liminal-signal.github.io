# Course Guide

## Policies

- Attend labs prepared and on time.
- Late work is accepted up to 48 hours with a 10 percent deduction.
- Collaboration is encouraged, but every write-up and Op-Ed is written individually.

## Grading

- Labs and checkpoints: 40 percent
- Writing and Op-Eds: 30 percent
- Final project and presentation: 30 percent

## Submissions

- All submissions happen through pull requests.
- Use one PR per assignment and include the assignment name in the title.
- Required reviews come from code owners or the editor team.
