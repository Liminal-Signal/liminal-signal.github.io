# Intro and Setup

This guide helps you get the repository running and make your first post.

## Checklist

- Clone the repo and install dependencies
- Run `mkdocs serve` to preview the site
- Create a branch and edit your post file
- Open a pull request for review

## First post workflow

1. Create a branch named `week-01/your-name`.
2. Edit your file in `docs/blog/posts/`.
3. Commit and push your changes.
4. Open a PR and request review.
